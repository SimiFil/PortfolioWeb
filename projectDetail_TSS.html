The Status Shuffle

The Status Shuffle is an app created by me (Filip Simandl) in 2025. It is a multiplayer party game designed for iOS devices, where players can connect and compete in real-time. More information can be found when scrolling down.

View Repository Live Demo

Overview

Social Ladder is an engaging iOS multiplayer party game that tests how well players know each other. Each round, one player becomes the "Leader" and must rank all participants based on a randomly selected question, while other players try to predict these rankings to earn points.

The game creates memorable moments of surprise, laughter, and discovery as players learn how their friends truly see them. Perfect for parties, game nights, or any social gathering!

Technology Used

The Status Shuffle is developed using Swift and SwiftUI, leveraging the power of Apple's frameworks to create a seamless and engaging user experience. The app utilizes GameKit for multiplayer functionality, allowing players to connect and compete in real-time.
